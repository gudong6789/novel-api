# Railway 部署指南

## 🚀 快速部署

### 方式一：使用 Railway GitHub 集成 (推荐)

1. **将代码推送到 GitHub**
```bash
cd C:\Users\dqb\.easyclaw\workspace
git add .
git commit -m "添加 Web 管理后台"
git push
```

2. **在 Railway 中重新部署**
   - 打开 Railway 项目
   - 点击 **Deployments**
   - 点击 **Deploy** 重新部署最新代码

### 方式二：使用 Railway CLI

1. **安装 Railway CLI**
```bash
npm install -g @railway/cli
```

2. **登录 Railway**
```bash
railway login
```

3. **部署项目**
```bash
cd C:\Users\dqb\.easyclaw\workspace\Server
railway up
```

## ️ 常见问题

### 问题 1: /admin 返回 404

**原因**: admin 文件夹未上传

**解决**:
```bash
# 检查文件是否存在
dir admin\index.html

# 重新推送代码
git add admin/
git commit -m "添加 admin 文件夹"
git push
```

### 问题 2: 数据库连接失败

**原因**: MongoDB 未配置

**解决**:
1. 在 Railway 面板添加 MongoDB 插件
2. 复制 `MONGODB_URI` 环境变量
3. 在 Railway 面板 → Variables 中添加

### 问题 3: 静态文件不加载

**原因**: 静态文件路径配置错误

**解决**: 确保 server.js 中有以下代码：
```javascript
app.use('/admin', express.static(path.join(__dirname, 'admin')));
```

## 📊 Railway 配置

### 环境变量

在 Railway 面板 → Variables 中配置：

| 变量名 | 值 |
|--------|-----|
| `MONGODB_URI` | Railway 自动提供 |
| `JWT_SECRET` | 随机字符串 |
| `ADMIN_USERNAME` | admin |
| `ADMIN_PASSWORD` | 你的密码 |
| `PORT` | 3000 |

### 启动命令

Railway 会自动检测 `package.json` 并运行：
```bash
npm start
```

如果没有自动检测，手动配置：
```bash
node server.js
```
