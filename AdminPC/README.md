# 小说自动阅读系统 - PC 管理后台

基于 Electron 开发的 Windows 桌面应用程序，提供完整的授权码和设备管理功能。

## 📦 项目结构

```
AdminPC/
├── package.json      # 项目配置
├── main.js           # Electron 主进程
├── preload.js        # 预加载脚本
├── index.html        # 主界面
├── styles.css        # 样式文件
├── renderer.js       # 渲染进程
└── README.md         # 本文件
```

## ✨ 功能特性

- 📊 **数据概览** - 实时统计授权码、设备、阅读时长数据
- 🎫 **授权码管理** - 单个/批量生成、封禁/解封、删除
- 📱 **设备管理** - 查看设备状态、阅读数据、最后活跃时间
- ⚙️ **全局配置** - 远程修改阅读客户端运行参数
- 🔧 **系统设置** - API 地址配置、软件信息
- 💾 **本地存储** - 记住登录状态、API 配置
- 🔔 **桌面通知** - 操作结果实时提醒

## 🚀 快速开始

### 方法一：使用已打包的应用程序 (推荐)

1. 下载打包好的安装包
2. 运行安装程序
3. 打开"小说管理后台"
4. 输入管理员账号登录

### 方法二：从源码运行

**前提条件:**
- Node.js 18+ (https://nodejs.org/)

**步骤:**

1. **安装依赖**
```bash
cd AdminPC
npm install
```

2. **开发模式运行**
```bash
npm start
```

3. **打包成可执行文件**
```bash
# 打包 Windows 安装程序
npm run build

# 打包便携版 (免安装)
npm run build:portable
```

打包后的文件在 `dist/` 目录。

## 📋 打包选项


```bash

```
生成：`dist/小说管理后台 x.x.x.exe`

## 🛠️ 开发指南

### 修改 API 地址

编辑 `main.js`:
```javascript
const API_BASE_URL = 'https://your-api-url.com/api';
```

### 修改主题颜色

编辑 `styles.css`:
```css
:root {
  --primary-color: #667eea;  /* 主色调 */
  --success-color: #48bb78;  /* 成功色 */
  --danger-color: #f56565;   /* 危险色 */
}
```

### 添加新功能

1. 在 `index.html` 添加 UI 元素
2. 在 `renderer.js` 添加逻辑处理
3. 在 `main.js` 添加 IPC 处理 (如需要)

## 🐛 常见问题

### Q: 运行时提示"无法连接到 API"

A: 检查网络连接，确认 API 地址配置正确。

### Q: 打包后程序无法启动

A: 
1. 确保所有依赖已安装
2. 检查 `package.json` 配置
3. 尝试重新打包

### Q: 如何修改应用图标？

A: 
1. 准备 `icon.ico` 文件 (256x256)
2. 放到项目根目录
3. 重新打包
