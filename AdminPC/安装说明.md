# 小说管理后台 - 安装说明

## ⚠️ 重要提示

由于 Electron 桌面应用需要下载较大的二进制文件（约 100MB），在当前环境下可能会遇到安装问题。

**推荐使用 Web 管理后台** - 功能完全相同，无需安装，打开浏览器即可使用！

## 📱 添加到浏览器快捷方式

### Chrome/Edge:

1. 打开 Web 管理后台
2. 点击右上角 **...** 菜单
3. 选择 **应用** → **将此站点作为应用安装**
4. 点击 **安装**
5. 桌面会自动创建快捷方式

### 优点:
- ✅ 独立窗口运行
- ✅ 无浏览器地址栏
- ✅ 类似原生应用体验
- ✅ 固定在任务栏

## 🖥️ 桌面版安装 (可选)

如果你确实需要桌面应用版本：

### 步骤 1: 安装 Node.js

下载地址：https://nodejs.org/
选择 LTS 版本安装

### 步骤 2: 安装依赖

打开命令行，运行：
```bash
cd C:\Users\dqb\.easyclaw\workspace\AdminPC
npm install
```

### 步骤 3: 启动应用

```bash
npm start
```

或双击运行：
```
启动小说管理后台.bat
```

### 步骤 4: 打包成安装包 (可选)

```bash
npm run build
```

生成的安装包在 `dist/` 目录。

## 🐛 故障排查

### 问题 1: 无法打开 Web 管理后台

**解决方法:**
1. 检查网络连接
2. 确认 API 服务正常运行
3. 尝试清除浏览器缓存

### 问题 2: 桌面版启动失败

**解决方法:**
```bash
# 重新安装依赖
cd C:\Users\dqb\.easyclaw\workspace\AdminPC
Remove-Item -Recurse -Force node_modules
npm install

# 启动
npm start
```

### 问题 3: 登录失败

**解决方法:**
1. 确认用户名密码正确
2. 检查 API 地址配置
3. 联系管理员重置密码
