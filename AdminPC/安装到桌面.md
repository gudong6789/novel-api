# 小说管理后台 - 桌面安装指南

## 🚀 快速启动

### 方式一：双击启动 (推荐)

直接双击运行：
```
C:\Users\dqb\.easyclaw\workspace\AdminPC\启动小说管理后台.bat
```

### 方式二：创建桌面快捷方式

1. **右键点击** `启动小说管理后台.bat`
2. 选择 **发送到** → **桌面快捷方式**
3. 桌面会出现快捷方式，双击即可启动

### 方式三：手动创建快捷方式

以管理员身份打开 PowerShell，运行：

```powershell
$WshShell = New-Object -comObject WScript.Shell
$Shortcut = $WshShell.CreateShortcut("$env:USERPROFILE\Desktop\小说管理后台.lnk")
$Shortcut.TargetPath = "C:\Users\dqb\.easyclaw\workspace\AdminPC\启动小说管理后台.bat"
$Shortcut.WorkingDirectory = "C:\Users\dqb\.easyclaw\workspace\AdminPC"
$Shortcut.IconLocation = "%SystemRoot%\System32\shell32.dll,13"
$Shortcut.Description = "小说自动阅读系统 - PC 管理后台"
$Shortcut.Save()
```

## 📦 打包成独立应用 (可选)

如果需要打包成独立的 .exe 安装程序：

### 步骤 1: 安装依赖

```bash
cd C:\Users\dqb\.easyclaw\workspace\AdminPC
npm install
```

### 步骤 2: 打包

```bash
npm run build
```

打包完成后在 `dist/` 目录会生成安装程序。

## 🔧 故障排查

### 问题 1: 提示找不到 electron

**解决方法:**
```bash
cd C:\Users\dqb\.easyclaw\workspace\AdminPC
npm install
```

### 问题 2: 首次启动慢

首次启动会下载 Electron 框架（约 100MB），请耐心等待。
后续启动会很快。

### 问题 3: 无法连接到 API

检查网络连接，确认 API 地址配置正确。
